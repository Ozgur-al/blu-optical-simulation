**Implementation strategy (both `_run_single` and `_trace_single_source` — they are mirrors):**

Step 1 — Compute axis-aligned bounding boxes for cylinder faces:
```python
# After existing solid_faces loop, add cylinder face AABBs
# Each CylinderCap is a disc -> AABB is a thin box of radius R centered at cap center
# Each CylinderSide wraps around cylinder -> AABB is box of (2R x 2R x length) centered at cylinder center
cyl_aabbs_list = []
for cface in cyl_faces:
    if isinstance(cface, CylinderCap):
        c = cface.center; r = cface.radius
        half = np.array([r, r, r])  # conservative: sphere-cap
        cyl_aabbs_list.append(np.concatenate([c - half, c + half]))
    else:  # CylinderSide
        c = cface.center; r = cface.radius; half_l = cface.length / 2.0
        # axis-aligned bounding box: expand in all 3 dims
        ax = np.abs(cface.axis)
        # box spans: in axis direction = ±half_l, in other two = ±r
        # Since axis may not be aligned, use conservative: (r + r + half_l) per dimension
        # More precise: project onto each axis
        half = np.array([
            r * (1 - ax[0]) + half_l * ax[0] + r,  # conservative expansion
            r * (1 - ax[1]) + half_l * ax[1] + r,
            r * (1 - ax[2]) + half_l * ax[2] + r,
        ])
        cyl_aabbs_list.append(np.concatenate([c - half, c + half]))
```

A simpler, guaranteed-correct conservative AABB for any cylinder face:
```python
import itertools
# Cylinder AABB: center ± (radius + half_length) in all dims (over-conservative but correct)
for cface in cyl_faces:
    if isinstance(cface, CylinderCap):
        c = cface.center; r = cface.radius
        # disc in plane perpendicular to normal — AABB is [c-r, c+r] in plane, thin in normal dir
        r_vec = np.array([r, r, r])
        n_abs = np.abs(cface.normal) * 0.001  # thin in normal direction
        lo = c - r_vec + n_abs * -1
        hi = c + r_vec
        cyl_aabbs_list.append(np.array([lo[0], lo[1], lo[2], hi[0], hi[1], hi[2]]))
    else:
        c = cface.center
        r = cface.radius; hl = cface.length / 2.0
        extent = r + hl  # over-conservative but always valid
        cyl_aabbs_list.append(np.array([c[0]-extent, c[1]-extent, c[2]-extent,
                                         c[0]+extent, c[1]+extent, c[2]+extent]))
```

Step 2 — Compute prism face AABBs:
```python
prism_aabbs_list = []
for pface in prism_faces:
    c = pface.center
    # Use center and half-size from vertices (most accurate)
    if hasattr(pface, 'vertices') and pface.vertices is not None:
        verts = np.array(pface.vertices)
        lo = verts.min(axis=0); hi = verts.max(axis=0)
        prism_aabbs_list.append(np.concatenate([lo, hi]))
    else:
        # Fallback: sphere bound around center
        extent = getattr(pface, 'circumscribed_radius', 10.0)
        prism_aabbs_list.append(np.array([c[0]-extent, c[1]-extent, c[2]-extent,
                                           c[0]+extent, c[1]+extent, c[2]+extent]))
```

The BVH `traverse_bvh_batch` returns `(t_arr, idx_arr)` where `idx` is the AABB index.
Currently we only call `traverse_bvh_batch` for plane surfaces — so the traverse result
is used to populate `best_t`/`best_type`/`best_obj`.

With extended AABBs, the BVH traverse still returns candidate indices. For cylinder/prism
indices (>= `n_all_planes`), the traverse tells us "this ray passes near this AABB" —
