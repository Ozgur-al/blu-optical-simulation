Create `backlight_sim/update_checker.py` -- a non-blocking update check module.

**Approach:** Use a simple JSON endpoint. For now, use GitHub Releases API (`https://api.github.com/repos/{owner}/{repo}/releases/latest`) as the check source, but abstract it so the URL is configurable. The module must work even if the repo doesn't exist yet -- it just returns "no update" on any failure.

**Classes/Functions:**

```python
@dataclass
class UpdateInfo:
    available: bool
    current_version: str
    latest_version: str = ""
    download_url: str = ""
    error: str = ""
```

`check_for_update(timeout: float = 5.0) -> UpdateInfo`:
- Uses `urllib.request.urlopen` with the specified timeout (default 5 seconds -- short enough for corporate networks)
- Parses JSON response, extracts `tag_name` (strip leading "v" if present)
- Compares versions using `packaging.version` if available, otherwise simple string split on "." and integer comparison
- Returns `UpdateInfo(available=True, ...)` if remote version is newer
- On ANY exception (timeout, DNS failure, SSL error, JSON parse error, HTTP error, proxy block): returns `UpdateInfo(available=False, error=str(e))` -- NEVER raises
- No PySide6 imports in this module (it can be called from a thread)

`check_for_update_async(callback, timeout: float = 5.0)`:
- Runs `check_for_update` in a `threading.Thread(daemon=True)`
- Calls `callback(UpdateInfo)` when done
- This is the preferred entry point from the GUI -- completely non-blocking

**Important:** This module uses only stdlib (`urllib.request`, `json`, `threading`, `dataclasses`). No external HTTP libraries needed. The PyInstaller .spec must un-exclude `http` and `urllib` for this to work in the frozen build.

**Version comparison fallback (no packaging dependency):**
```python
def _compare_versions(current: str, latest: str) -> bool:
    """Return True if latest > current."""
    try:
        c = tuple(int(x) for x in current.split("."))
        l = tuple(int(x) for x in latest.split("."))
        return l > c
    except (ValueError, AttributeError):
        return False
```

```python
    samples_dir = DIST_DIR / "samples"
```

```python
"""Generate sample .blu project files for distribution."""
import sys
from pathlib import Path

# Add project root to path so we can import backlight_sim
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))

from backlight_sim.io.presets import preset_simple_box, preset_automotive_cluster, preset_edge_lit_lgp
from backlight_sim.io.project_io import save_project

def main():
    output_dir = Path(sys.argv[1]) if len(sys.argv) > 1 else Path(".")
    output_dir.mkdir(parents=True, exist_ok=True)

    samples = [
        ("Simple_Box_Demo.blu", preset_simple_box),
        ("Automotive_Cluster_Demo.blu", preset_automotive_cluster),
        ("Edge_Lit_LGP_Demo.blu", preset_edge_lit_lgp),
    ]
    for filename, factory in samples:
        project = factory()
        save_project(project, str(output_dir / filename))
        print(f"  Created {filename}")

if __name__ == "__main__":
    main()
```
