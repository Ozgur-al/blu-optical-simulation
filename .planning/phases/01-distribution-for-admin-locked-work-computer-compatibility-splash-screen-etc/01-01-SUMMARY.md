## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Fixed Pillow ICO multi-size save ordering**
- **Found during:** Task 2 (Generate application icon)
- **Issue:** Initial Pillow save used 16px as anchor image with `sizes=` kwarg, producing a single-frame 548-byte ICO instead of 6-frame file
- **Fix:** Changed to anchor=256px image with descending `append_images`; removed `sizes=` kwarg (incompatible with current Pillow)
- **Files modified:** assets/icon.py
- **Verification:** PIL IcoImagePlugin confirms 6 sizes present; file size 38,099 bytes
- **Committed in:** 1f0266e (Task 2 commit)

**2. [Rule 1 - Bug] Fixed Unicode arrow in print statement**
- **Found during:** Task 2 (running icon.py on Windows with cp1252 console)
- **Issue:** `->` written as `->` unicode arrow caused UnicodeEncodeError on Windows cp1252 console
- **Fix:** Replaced unicode arrow with ASCII `->` in print statements
- **Files modified:** assets/icon.py
- **Verification:** Script runs without error on Windows
- **Committed in:** 1f0266e (Task 2 commit)
