       from __future__ import annotations
       from dataclasses import dataclass, field
       import numpy as np
       from backlight_sim.core.geometry import Rectangle

       FACE_NAMES = ("top", "bottom", "left", "right", "front", "back")

       @dataclass
       class SolidBox:
           name: str
           center: np.ndarray
           dimensions: tuple[float, float, float]  # (W, H, D)
           material_name: str = "pmma"
           face_optics: dict[str, str] = field(default_factory=dict)
           coupling_edges: list[str] = field(default_factory=list)

           def __post_init__(self):
               self.center = np.asarray(self.center, dtype=float)

           def get_faces(self) -> list[Rectangle]:
               W, H, D = self.dimensions
               cx, cy, cz = self.center
               hw, hh, hd = W / 2.0, H / 2.0, D / 2.0
               face_specs = {
                   "top":    (np.array([cx, cy, cz + hd]), 2,  1.0, (W, H)),
                   "bottom": (np.array([cx, cy, cz - hd]), 2, -1.0, (W, H)),
                   "right":  (np.array([cx + hw, cy, cz]), 0,  1.0, (H, D)),
                   "left":   (np.array([cx - hw, cy, cz]), 0, -1.0, (H, D)),
                   "back":   (np.array([cx, cy + hh, cz]), 1,  1.0, (W, D)),
                   "front":  (np.array([cx, cy - hh, cz]), 1, -1.0, (W, D)),
               }
               faces = []
               for face_id, (center, axis, sign, size) in face_specs.items():
                   rect = Rectangle.axis_aligned(
                       name=f"{self.name}::{face_id}",
                       center=center, size=size,
                       normal_axis=axis, normal_sign=sign,
                       material_name=self.material_name,
                   )
                   rect.optical_properties_name = self.face_optics.get(face_id, "")
                   faces.append(rect)
               return faces
       ```

    1. **Add helper functions** (module-level, before the RayTracer class):

       `_fresnel_unpolarized(cos_theta_i, n1, n2)` — Vectorized unpolarized Fresnel reflectance. Takes (N,) arrays of cos_theta_i, n1, n2. Returns (N,) R values in [0,1]. TIR case (sin_theta_t_sq >= 1) returns 1.0. Formula: compute sin_theta_t_sq = (n1/n2)^2 * (1 - cos_i^2), cos_theta_t = sqrt(max(0, 1 - sin_theta_t_sq)), then Rs = ((n1*cos_i - n2*cos_t)/(n1*cos_i + n2*cos_t))^2, Rp = ((n2*cos_i - n1*cos_t)/(n2*cos_i + n1*cos_t))^2, R = 0.5*(Rs+Rp). Guard denominators with max(..., 1e-12). Use np.where for TIR mask.

       `_refract_snell(directions, oriented_normals, n1, n2)` — Vectorized Snell's law refraction. Takes (N,3) directions, (N,3) oriented_normals (pointing INTO the new medium), (N,) n1, (N,) n2. Returns (N,3) refracted unit directions. Formula: eta=n1/n2, cos_i=clip(-dot(d,n), 0, 1), sin_t_sq=eta^2*(1-cos_i^2), cos_t=sqrt(max(0, 1-sin_t_sq)), refracted = eta*d + (eta*cos_i - cos_t)*n. Normalize output.

    2. **Modify `_run_single()`:**

               solid_faces.append(face)

       ```

       Initialize per-ray `current_n = np.ones(n_rays, dtype=float)` alongside origins/directions/weights.

       Initialize solid_body_stats accumulator:
       ```python
       sb_stats = {}
       for box in self.project.solid_bodies:
           sb_stats[box.name] = {fid: {"entering_flux": 0.0, "exiting_flux": 0.0} for fid in FACE_NAMES}
       ```

       In the bounce loop intersection section, add solid_faces to the intersection candidates alongside regular surfaces. Use a type flag to distinguish: surfaces get normal material behavior, solid_faces get Fresnel physics.

       **For SolidBox face hits:**
       - Compute oriented normal `on`: if dot(direction, face.normal) < 0, ray is entering (on = face.normal); else exiting (on = -face.normal).
       - Entry: n1 = current_n[hit_idx], n2 = box_n. Update current_n[hit_idx] = box_n.
       - Exit: n1 = current_n[hit_idx], n2 = 1.0. Update current_n[hit_idx] = 1.0.
       - Compute cos_theta_i = clip(-dot(direction, on), 0, 1).
       - Compute R = _fresnel_unpolarized(cos_theta_i, n1, n2).
       - Stochastic Russian roulette: roll = rng.random(len(hit_idx)). If roll < R → reflect (specular: d' = d - 2*(d.n)*n). Else → refract (use _refract_snell).
       - Ray keeps full weight (energy-conserving).
       - Accumulate flux in sb_stats: entering_flux += weight for entry, exiting_flux += weight for exit.
       - **CRITICAL: Use geometry-relative epsilon** for origin offset: `geom_eps = max(1e-6, min(box.dimensions) * 1e-4)`. Offset: `origins[hit_idx] = hit_point + on * geom_eps`. Do NOT use the global `_EPSILON` for SolidBox faces.
