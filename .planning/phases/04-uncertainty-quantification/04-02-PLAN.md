<objective>
Wire batch emission into the tracer so that every simulation run populates per-batch detector grids on DetectorResult without modifying the C++ extension.

Purpose: Wave 1 defined the data model (including DetectorResult.rays_per_batch and SimulationResult.uq_warnings) and CI math; Wave 2 produces the raw data those helpers consume. The tracer must split rays_per_source into K batches, snapshot each batch's grid separately, and return K grids on DetectorResult.grid_batches (plus hits_batches, flux_batches, rays_per_batch, and optionally grid_spectral_batches). This has to work across all three tracer paths (C++ fast path, Python fallback path, multiprocessing path) with deterministic per-batch RNG seeding.

Output:
- `_run_single` slices its inner batch loop into K chunks of size `rays_per_source // K` (with remainder handling), copying `grid` and `grid_spectral` snapshots into per-batch accumulators. Adaptive convergence is evaluated only at chunk boundaries per CONTEXT D-01.
- `_cpp_trace_single_source` loops K times over the existing `_blu_tracer.trace_source(project_dict, source_name, seed)` with per-batch hashed seeds (`md5(f"{base_seed}_{source_name}_{k}")`).
- `_run_multiprocess` merges per-batch grids (and rays_per_batch) from each worker's return dict into the aggregate DetectorResult.
- `uq_batches=0` opt-out preserves the exact pre-Phase-4 behavior (determinism anchor).
- Adaptive sampling + UQ runs both simultaneously (per CONTEXT D-01): tracer appends a warning string to SimulationResult.uq_warnings (field already defined in Wave 1) so Wave 3 UI can display it. Adaptive convergence evaluation is gated to chunk boundaries to preserve Student-t validity.
- `uq_include_spectral=False` skips spectral per-batch caching (memory knob).
- Path recording honors first-batch-only rule: only k=0 records rays_paths; subsequent batches do not inflate memory.
- DetectorResult.rays_per_batch is populated with the actual per-batch ray counts (accounts for remainder distribution); consumed downstream by Wave 3 for unbiased per-batch efficiency scaling.
- New tests: legacy-equivalence (K=0 bit-match), ray-count conservation (K × batch_rays == rays_per_source), 1/sqrt(N) shrinkage on Lambertian reference, MP parity, C++-vs-Python stderr agreement, adaptive-early-convergence-at-chunk-boundary.

    Step A — Add a helper near the top of `backlight_sim/sim/tracer.py` (after imports):

    ```python
    import hashlib

    def _effective_uq_batches(settings) -> int:
        """Return effective K honoring the [4, 20] dof window when UQ is on.

        Returns 0 when UQ disabled (uq_batches <= 0).
        Clamps: K = min(20, max(4, uq_batches)) when uq_batches > 0.
        """
        k_req = int(getattr(settings, "uq_batches", 0))
        if k_req <= 0:
            return 0
        return min(20, max(4, k_req))


    def _batch_seed(base_seed: int, source_name: str, batch_k: int) -> int:
        """Deterministic per-batch seed: md5(f'{base}_{source}_{k}')[:8] -> int."""
        h = hashlib.md5(f"{base_seed}_{source_name}_{batch_k}".encode("utf-8")).hexdigest()
        return int(h[:16], 16) & 0xFFFFFFFFFFFFFFFF  # 64-bit mask


    def _partition_rays(rays_total: int, K: int) -> list[int]:
        """Split rays_total into K batches; distribute remainder to the first (rays_total % K) batches.

        Returns list of length K. sum(result) == rays_total. Used for both rays_per_batch
        bookkeeping and the actual per-batch rays_per_source value passed to the tracer.
        """
        base = rays_total // K
        rem = rays_total - base * K
        return [base + (1 if k < rem else 0) for k in range(K)]
    ```

    ```python
    def _cpp_trace_single_source(project, source_name: int, base_seed: int) -> dict:
        project_dict = _serialize_project(project)
        settings = project.settings
        K = _effective_uq_batches(settings)

        if K == 0:
            # Legacy fast path — single call, no UQ data.
            result_dict = _blu_tracer.trace_source(project_dict, source_name, base_seed)
            result_dict["grids_batches"] = None
            result_dict["hits_batches"] = None
            result_dict["flux_batches"] = None
            result_dict["rays_per_batch"] = None
            return result_dict

        # UQ path: K sequential calls, per-batch seed + sliced rays_per_source.
        rays_total = int(settings.rays_per_source)
        rays_per_batch = _partition_rays(rays_total, K)

        per_batch_dicts: list[dict] = []
        actual_rays_per_batch: list[int] = []
        for k in range(K):
            rays_this_batch = rays_per_batch[k]
            if rays_this_batch <= 0:
                continue
            batch_project_dict = dict(project_dict)  # shallow
            batch_project_dict["settings"] = dict(project_dict["settings"])
            batch_project_dict["settings"]["rays_per_source"] = rays_this_batch
            seed_k = _batch_seed(base_seed, source_name, k)
            batch_result = _blu_tracer.trace_source(batch_project_dict, source_name, seed_k)
            per_batch_dicts.append(batch_result)
            actual_rays_per_batch.append(rays_this_batch)

        # Merge: sum grids across batches, stack per-batch arrays.
        merged = _merge_cpp_batches(per_batch_dicts, K=len(per_batch_dicts))
        merged["rays_per_batch"] = actual_rays_per_batch
        return merged
    ```
