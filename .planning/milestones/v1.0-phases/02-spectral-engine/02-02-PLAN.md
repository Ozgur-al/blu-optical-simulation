  <name>Task 1: CIE colorimetry helpers and Spectral Data panel</name>
  <files>backlight_sim/sim/spectral.py, backlight_sim/gui/spectral_data_panel.py, backlight_sim/gui/main_window.py</files>
  <action>
    **1. CIE colorimetry helpers** (`sim/spectral.py`) — add the following functions:

    - `xyz_per_pixel(spectral_grid, wavelengths)`: Takes `(ny, nx, n_bins)` spectral grid and `(n_bins,)` wavelengths. Builds CIE XYZ weight matrix `(n_bins, 3)` by interpolating `_CIE_X/Y/Z` at each wavelength. Returns `spectral_grid @ xyz_weights` as `(ny, nx, 3)` XYZ image.

    - `xy_per_pixel(xyz)`: CIE 1931 (x, y) chromaticity from `(ny, nx, 3)` XYZ. `x = X/(X+Y+Z)`, `y = Y/(X+Y+Z)`. Guard zero-sum pixels with `np.where(s > 0, s, 1.0)`. Returns `(ny, nx, 2)`.

    - `uv_per_pixel(xyz)`: CIE 1976 u'v' from XYZ. `u' = 4X/(X+15Y+3Z)`, `v' = 9Y/(X+15Y+3Z)`. Returns `(ny, nx, 2)`.

    - `cct_robertson(xy)`: Robertson (1968) CCT from (x, y) chromaticity. Inline the 31-entry isotherm table (u_i, v_i, t_i, slope_i for CCT values 1000 K through infinity). Convert xy to CIE 1960 UCS `(u, v)` for the isotherms: `u = 4x/(-2x+12y+3)`, `v = 6y/(-2x+12y+3)`. Find enclosing isotherm pair by sign change of distance `d_i = (v - v_i) - slope_i * (u - u_i)`. Interpolate CCT. Clamp to [1000, 25000] K range. Return NaN for pixels with Y < 1e-10. Return array matching input shape minus last dim.

    - `compute_color_kpis(spectral_grid, wavelengths)`: Master function that computes all color KPIs. Returns a dict:
      ```python
      {
          "delta_ccx": float, "delta_ccy": float,        # CIE 1931 full detector
          "delta_uprime": float, "delta_vprime": float,   # CIE 1976 full detector
          "cct_avg": float, "cct_range": float,           # CCT full detector
          "center_1_4": {"delta_ccx": ..., "delta_ccy": ..., "delta_uprime": ..., "delta_vprime": ...},
          "center_1_6": {"delta_ccx": ..., "delta_ccy": ..., ...},
          "center_1_10": {"delta_ccx": ..., "delta_ccy": ..., ...},
      }
      ```
      For center fractions, use `f_side = sqrt(fraction)`, crop to center region, compute delta of xy and u'v' within that region. Only include pixels with non-zero luminance (Y > threshold).

    **2. Spectral Data panel** (`gui/spectral_data_panel.py`) — NEW FILE:

    Create `SpectralDataPanel(QWidget)` with two sub-sections in a `QSplitter` (vertical):

    **Top section: Source SPD Manager**
    - `QComboBox` selector for SPD profiles (populated from `project.spd_profiles` keys + built-in names as read-only entries)
    - Buttons: Import CSV, Export CSV, Delete, Duplicate, Blackbody Generator
    - `QTableWidget` with columns: wavelength_nm, intensity — editable for custom profiles, read-only for built-ins
    - Blackbody generator: `QDialog` with `QDoubleSpinBox` for CCT (range 1000-25000, default 5000), generates SPD via `blackbody_spd()`, adds to project.spd_profiles with name "blackbody_{cct}K"
    - `pyqtgraph.PlotWidget` showing SPD curve (wavelength vs intensity)

    **Bottom section: Material Spectral Tables**
    - `QComboBox` selector for material/optical_properties names (from project.materials + project.optical_properties)
    - Buttons: Import CSV, Export CSV, Clear
    - `QTableWidget` with columns: wavelength_nm, reflectance, transmittance — editable
    - `pyqtgraph.PlotWidget` showing R(lambda) and T(lambda) curves

    **Right panel: Chromaticity Diagram** (in a `QSplitter` horizontal with the table area on left, diagram on right)
    - `pyqtgraph.PlotWidget` with:
      - CIE 1931 spectral locus drawn as `PlotCurveItem` from `_CIE_X/Y/Z` (compute `x = X/(X+Y+Z)`, `y = Y/(X+Y+Z)`, close the locus with line of purples)
      - Planckian locus as a second curve (compute from blackbody xy at CCTs 1000-25000 K)
      - Scatter overlay of per-pixel chromaticity points (populated after simulation via `update_chromaticity(spectral_grid)`)
      - Subsample chromaticity points: use `xy[::4, ::4]` to limit to ~625 points max for a 100x100 detector
      - Axis labels "x" and "y", title "CIE 1931"

    **Signals and wiring:**
    - `spectral_data_changed = Signal()` — emitted on any edit to SPD or material spectral tables
    - `set_project(project)` — called by MainWindow to load project data
    - `update_chromaticity(spectral_grid)` — called after simulation to update scatter plot

    Follow the `AngularDistributionPanel` pattern: `_loading_table` guard, `blockSignals()` on combo changes, CSV import via `QFileDialog`, normalize buttons (peak=1) for SPD profiles.
