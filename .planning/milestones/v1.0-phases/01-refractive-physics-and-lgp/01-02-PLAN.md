    2. Add the following logic after the existing energy balance rows:

    ```python
    # --- LGP metrics (only if solid bodies exist) ---
    if hasattr(result, 'solid_body_stats') and result.solid_body_stats and project.solid_bodies:
        # Compute coupling efficiency: flux entering through coupling faces / total emitted
        total_coupling_flux = 0.0
        for box in project.solid_bodies:
            box_stats = result.solid_body_stats.get(box.name, {})
            for edge_id in box.coupling_edges:
                face_data = box_stats.get(edge_id, {})
                total_coupling_flux += face_data.get("entering_flux", 0.0)

        total_emitted = result.total_emitted_flux
        coupling_eff = total_coupling_flux / total_emitted if total_emitted > 0 else 0.0

        # Extraction efficiency: detector flux / coupling flux
        det_flux = sum(dr.total_flux for dr in result.detectors.values())
        extraction_eff = det_flux / total_coupling_flux if total_coupling_flux > 0 else 0.0

        # Overall efficiency: coupling * extraction
        overall_eff = coupling_eff * extraction_eff
    ```

    3. Display three new rows in the energy balance section of the KPI table:
       - "Edge Coupling Eff." — `f"{coupling_eff*100:.1f}%"` — tooltip: "Fraction of emitted flux entering LGP through coupling edge(s)"

       - "Overall LGP Eff." — `f"{overall_eff*100:.1f}%"` — tooltip: "End-to-end efficiency (coupling x extraction)"
