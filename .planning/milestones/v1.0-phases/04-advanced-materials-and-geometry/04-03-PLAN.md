       - Vectorized over N rays using numpy
       - Compute components perpendicular to axis: `d_perp = d - (d.axis)*axis`, `oc_perp = (o-center) - ((o-center).axis)*axis`
       - Quadratic: `a = |d_perp|^2`, `b = 2*(d_perp . oc_perp)`, `c = |oc_perp|^2 - R^2`
       - Solve discriminant, pick nearest positive t > epsilon
       - Check height constraint: `|(hit - center) . axis| <= half_length`
       - Return (N,) t values (np.inf for miss)

    2. Add `_intersect_rays_disc(origins, directions, center, normal, radius)` to `sim/tracer.py`:
       - Ray-plane intersection (reuse math from _intersect_rays_plane but with circular boundary)
       - `t = dot(center - o, normal) / dot(d, normal)`, check t > epsilon
       - `hit = o + t*d`, check `||hit - center||^2 <= radius^2`
       - Return (N,) t values

    3. Add `_intersect_prism_cap(origins, directions, cap)` to `sim/tracer.py`:
       - First: ray-plane intersection with the cap plane (same as disc but polygon boundary)
       - Then: point-in-polygon test using precomputed edge normals from the PrismCap object
       - For hit point (u, v) in local cap coords: check all N signed distances `dot(p - vertex[i], edge_normal[i]) >= 0` (convex containment)
       - Return (N,) t values
