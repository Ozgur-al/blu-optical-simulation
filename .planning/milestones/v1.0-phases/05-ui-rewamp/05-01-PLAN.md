  1. Reads `dark.qss` from same directory using `os.path.dirname(__file__)`
  2. Calls `app.setStyleSheet(qss_content)`
  3. Sets pyqtgraph config: `pg.setConfigOption('background', BG_BASE)`, `pg.setConfigOption('foreground', TEXT_PRIMARY)`, `pg.setConfigOption('antialias', True)` — NOTE: use `pg.setConfigOption` (singular), NOT `pg.setConfigOptions` (plural, silently ignored)

**`dark.qss`** must cover these widget types with the dark palette:
- `QMainWindow` (background)
- `QDockWidget` (background, title bar with darker shade, no visible borders — use 2-3px gap via margin)
- `QDockWidget::title` (background slightly lighter than panel, padding)
- `QTreeWidget`, `QTreeWidget::item`, `QTreeWidget::item:selected` (teal accent background on selection)
- `QScrollArea`, `QScrollBar:vertical`, `QScrollBar:horizontal` (dark scrollbars with muted handles)
- `QGroupBox` (panel background, title styling)
- `QLineEdit`, `QDoubleSpinBox`, `QSpinBox`, `QComboBox` (recessed/inset style — background darker than panel `#181818`, subtle 1px border `#333`)
- `QDoubleSpinBox::up-button`, `QDoubleSpinBox::down-button`, `QDoubleSpinBox::up-arrow`, `QDoubleSpinBox::down-arrow` (visible arrows on dark background — use border-image or explicit color)
- `QPushButton` (default dark style + teal accent variant via `QPushButton[accent="true"]`)
- `QStatusBar` (dark background)
- `QMenuBar`, `QMenu`, `QMenu::item:selected` (dark with teal hover)
- `QToolBar`, `QToolButton` (dark, icon+text spacing)
- `QTabBar::tab`, `QTabBar::tab:selected` (if any remaining tab widgets)
- `QSplitter::handle` (narrow, subtle)
- `QTextEdit` (for log dock — monospace, dark background)
- `QCheckBox`, `QRadioButton` (visible indicators on dark)
- `QLabel` (default text color)
- `QProgressBar` (teal fill on dark track)

Use the borderless panel separator approach per user decision: `QDockWidget { margin: 1px; }` instead of visible borders. Use recessed inputs per user decision: inputs darker than panel background with subtle inner shadow effect via `border: 1px solid #333; background: #181818;`.
