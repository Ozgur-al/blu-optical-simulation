```python
def _threshold_color(value: float, green_thresh: float, yellow_thresh: float, higher_is_better: bool = True) -> str:
    """Return CSS color string for green/yellow/red threshold."""
    if higher_is_better:
        if value >= green_thresh: return "color: #4caf50;"   # green
        elif value >= yellow_thresh: return "color: #ff9800;" # orange
        else: return "color: #f44336;"                        # red
    else:

        else: return "color: #f44336;"
```

3. In `_run_single()`, track emit progress and emit partial results at 5% intervals:
   - Add `last_emit_progress = 0.0` at the start
   - After each source completes (where `progress_callback` is called), check if current progress exceeds `last_emit_progress + 0.05`
   - If yes, build a lightweight `SimulationResult` snapshot:
     ```python
     if partial_result_callback and (progress - last_emit_progress) >= 0.05:
         # Snapshot detector grids (shallow copy — grid.copy() is fast)
         partial_detectors = {}
         for det_name, det_result in detector_results.items():
             partial_detectors[det_name] = DetectorResult(
                 detector_name=det_name,
                 grid=det_result.grid.copy(),
                 total_hits=det_result.total_hits,
                 total_flux=det_result.total_flux,
             )
         partial = SimulationResult(
             detectors=partial_detectors,
             ray_paths=[],
             escaped_flux=escaped_flux,
             total_emitted_flux=total_emitted,
             source_count=len(sources),
         )
         partial_result_callback(partial)
         last_emit_progress = progress
     ```
   - Use `grid.copy()` (numpy shallow copy producing new array) — NOT `copy.deepcopy` which is too expensive at 5% intervals
   - Do NOT include ray_paths, sphere_detectors, or solid_body_stats in partial results — just detector grids and energy balance

4. **Multiprocessing guard:** In `_run_multiprocess()`, do NOT pass `partial_result_callback`. In `run()`, when choosing MP path, pass `partial_result_callback=None` to the MP function. Live preview only works in single-thread mode.

5. Make sure the existing `_run_single` signature accepts `partial_result_callback` as a keyword argument with default `None` so existing callers (parameter sweep, etc.) are not broken.
